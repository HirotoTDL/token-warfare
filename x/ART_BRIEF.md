# TOKEN WARFARE アート発注書(codex → GPT画像生成用)

このセッション(Claude)が発注と品質管理を担当し、生成はcodex経由でGPTに依頼する。
**現状このPCにcodex CLIは未導入**。導入後、本ブリーフのプロンプトを1件ずつ流すこと。

## 運用ルール

- **1枚ずつ個別生成。グリッドシート(複数画像を1枚に並べて分割)は禁止**(ユーザー方針・品質劣化のため)。
- 生成物の保存先: `token-warfare/art/raw/`(原版)→ 採用版を `token-warfare/public/art/` に配置。
- 品質基準(Claude側でチェックする項目):
  1. イメージワード「パンク・ストリート・ビビット・ポップ・かっこいい・親しみ」に合致
  2. ゲーム内視認性(小さく表示しても誰/何か分かる)
  3. チームカラー(青/赤)を将来差し替え可能な配色設計(キャラ固有色と衝突しない)
  4. シリーズ間の画風統一(同一スタイル指定文を全プロンプトに含める)

## 共通スタイル指定(全プロンプト先頭に付ける)

> ポップでビビットなストリート/パンクアートスタイル。太いアウトライン、彩度の高い配色(ホットピンク #ff4fa3 / シアン #29d3e8 / イエロー #ffd23e / パープル #9b5cff)、グラフィティとスケートカルチャーの空気感。可愛さと洗練の両立。武器は鉄の銃ではなくポップアートのようなエネルギーブラスター。

## ゲームへの組み込み手順(生成後にClaudeが実施)

1. `public/art/` に配置し、`models.ts`/`arena.ts` のプロシージャルテクスチャを `TextureLoader` 読み込みに差し替え
2. タイトル画面ロゴ(`index.html` の .logo)を画像に差し替え
3. キャラ選択カードに立ち絵を表示(`main.ts` のカード生成部)
4. 品質チェック(視認性・統一感)→ NGなら本ブリーフの該当プロンプトを修正して再発注
