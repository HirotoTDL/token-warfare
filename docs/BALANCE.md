# バランス記録

## 検証手法

ヘッドレス自動対戦シミュレータ(`src/sim.ts`)でボット同士(Lv6)の3分マッチを全キャラ総当たり。
コンソールから実行: `__tw.sim('renji','jin')` / `__tw.simMatrix(6).then(r => __tw.simSummary(r))`

**注意: ボット自己対戦は人間プレイの近似でしかない**(人間は機動・エイム・読みで近距離型を活かせる)。
シムは「構造的な歪み」の検出に使い、最終調整は実プレイのフィードバックで行う。

**診断**: 遠距離型が過剰。原因は武器値よりボットの立ち回り(開けた長射線で誰とでも撃ち合う)。

### 適用した調整
1. 武器: ジン微ネルフ(rate 0.9→0.82, 遠距離倍率1.3→1.2)、ユメ微ネルフ(10→9.5)、
   ガロ微バフ(中距離0.5→0.58, 燃費13→12)、ミミバフ(HP100→108, 4.6→4.8, 減衰緩和)
2. **ボット行動**: 武器のピーク距離帯のカバーを陣取る/威力が出ない距離では足を止めず詰める
   (超近距離型は威力80%圏まで我慢)
